Serverless Web App

오후의 식물 (Plantnoon)

식집사를 위한 AI 진단 및 토탈 관리 플랫폼입니다.
별도의 백엔드 서버 구축 없이 Firebase Serverless 아키텍처를 활용하여 사용자 인증, 데이터베이스, 파일 저장소 기능을 구현했습니다.
Tech: JavaScript, Firebase(Firestore, Auth, Storage), PortOne, Gemini API
Type: Serverless Architecture, SPA
  • Serverless 아키텍처: Firebase Authentication, Firestore NoSQL DB 설계
  • 결제 프로세스 구현: PortOne SDK 연동 및 구독(Pro) 등급 권한 관리
  • AI 서비스 연동: Google Gemini API를 활용한 식물 진단 챗봇
  • 비동기 데이터 처리: Fetch API 및 Async/Await을 활용한 SPA 환경 구축
